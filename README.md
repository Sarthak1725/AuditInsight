# AuditInsight – AI-Powered Risk & Compliance Analytics Tool

## Overview

This project implements an end-to-end AI-powered tool for detecting financial anomalies, automating compliance checks, and providing risk insights using machine learning.

## Features

- Data ingestion and preprocessing
- Automated compliance checks
- Anomaly detection with Isolation Forest and DBSCAN
- NLP-based risk classification
- Risk scoring system
- Interactive dashboards (PowerBI)
- Audit trail logging

## Technology Stack

- Python
- Pandas, Scikit-learn, SpaCy
- Jupyter Notebook
- PowerBI, Excel

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Download SpaCy model: `python -m spacy download en_core_web_sm`
3. Run the notebooks or scripts as needed.

## Project Structure

- data/: Datasets
- models/: Trained models
- notebooks/: Jupyter notebooks
- scripts/: Python scripts
- docs/: Documentation
- tests/: Unit tests
